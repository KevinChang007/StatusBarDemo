# StatusBarDemo
沉浸式状态栏，支持设置状态栏背景和颜色，修复其他项目出现的键盘弹起遮挡输入框问题，以及5.1 的设备CoordinatorLayout +AppBarLayout 滑动问题  

Download sample [Apk] (https://github.com/LuckyJayce/StatusBarDemo/blob/master/raw/StatusBarDemo.apk?raw=true)
